🧠 Butler — Your Personal AI Assistant
Butler is an intelligent personal assistant powered by multiple AI systems. Designed to streamline your digital life, it can handle a wide variety of tasks based on your commands—ranging from information retrieval to automation, creative writing, scheduling, and more.

🔍 Features
🗣️ Natural Language Understanding
Communicate with Butler using plain language.

🧠 Multi-AI Core
Integrates different AI models and tools depending on the task type (e.g., LLMs, speech models, image processing).

🛠️ Modular Task Engine
Perform tasks like:

Writing & editing content

Summarizing articles & PDFs

Generating images or code

Managing files, reminders, or notes

Executing shell/system commands (optional)

More via plug-ins

🔄 Memory & Context Awareness (optional)
Can remember user preferences and past tasks if configured.

🔐 Privacy First
Designed to run locally or with optional cloud APIs. Your data is yours.

🧠 AI Backends Supported
OpenAI (ChatGPT, Whisper, DALL·E)

Ollama (e.g., Mistral, LLaMA)

Local LLMs via Transformers

Stable Diffusion (via API or local)

Others via plugin adapters

🔌 Plugin Examples
weather: Get local forecasts

summarize: Summarize text or documents

search: Real-time web search

notes: Create & manage personal notes

exec: Shell command execution (guarded)
